# AURA 프로젝트 진행 현황

## 개요
- **프론트엔드(aura-client)**: React + TypeScript + Vite 기반. 현재 로그인/회원가입 단일 화면을 구현해 디자인을 적용한 상태입니다.
- **백엔드(aura-backend)**: Spring Boot 3.5.8 기반의 초기 구조만 구성되어 있으며, PostgreSQL 연동 설정과 JPA/보안 의존성만 추가된 상태입니다.

## 프론트엔드 개발 현황
- 구현된 화면: 로그인/회원가입 전환이 가능한 인증 랜딩 화면.
- 주요 특징:
  - 상단 인트로 섹션과 탭 전환이 가능한 인증 패널 레이아웃.
  - 기본 로그인 폼(아이디/비밀번호, 아이디 저장 체크박스)과 소셜 로그인 버튼(카카오, 네이버, 구글) 배치.
  - 계정 복구(아이디/비밀번호 찾기) 버튼 그룹 제공.
  - 회원가입 패널에서는 소셜 간편 가입 버튼과 로그인 이동 링크 제공.
  - 다크 톤 그라데이션 배경과 카드 스타일 패널, 반응형 그리드 레이아웃으로 구성.
- 추가 메모: 브라우저 스크린샷 시도는 Playwright 환경 오류로 실패했습니다. (포트 4173에서 Vite dev 서버는 정상 기동 확인)

## 백엔드 개발 현황
- 현재 `BackendApplication` 진입점만 존재하며 별도의 도메인/컨트롤러/서비스 구현은 없음.
- PostgreSQL 데이터소스 설정 및 Hibernate DDL auto-update 옵션이 적용되어 초기 개발 준비 상태입니다.
- Lombok, Spring Web/Security/Data JPA/Validation 의존성만 추가된 기본 뼈대 단계입니다.

## 현재까지 구현된 페이지 모습(설명)
- 인트로 카피: "당신만의 공간을 더 선명하게 열어보세요" 문구와 서비스 가치 Pill 태그(간편 로그인, 보안 강화, 다중 인증 지원).
- 인증 패널:
  - **로그인 탭**: 아이디/비밀번호 입력, 아이디 저장 체크박스, 기본 로그인 버튼.
  - **소셜 로그인 영역**: 카카오/네이버/구글 버튼(각 서비스 색상 반영).
  - **계정 복구 영역**: 아이디 찾기/비밀번호 찾기 버튼(그라데이션 강조색 적용).
  - 탭 전환을 통해 **회원가입 탭**으로 이동 가능하며, 소셜 간편 가입 버튼과 로그인 이동 링크 제공.
